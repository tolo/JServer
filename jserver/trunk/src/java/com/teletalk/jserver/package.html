Provides classes that form the very base of the JServer architechture. The most important class in this package is {@link com.teletalk.jserver.SubSystem SubSystem}, which plays a key role in adding new functionality to the server. This package also contains the top (or root, depending how you look at it) system of the server, the {@link com.teletalk.jserver.JServer JServer} class, which is used when creating a server. It is the main class of the server.
